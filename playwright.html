The DNA of a Test

Beginner Fundamentals

Every Playwright test follows a simple pattern: Arrange, Act, Assert.

import { test, expect } from '@playwright/test';

test('has title and link', async ({ page }) => {
  // 1. Arrange: Go to the page
  await page.goto('https://playwright.dev/');

  // 2. Act: Click the 'Get Started' link
  await page.getByRole('link', { name: 'Get started' }).click();

  // 3. Assert: Verify the URL contains 'intro'
  await expect(page).toHaveURL(/.*intro/);
});

The 'page' Fixture

Provides isolated browser contexts for every test. No cookies or cache leakage.

Auto-Waiting

Actions like .click() automatically wait for elements to be visible and stable.

Precision Locators

Level 2: The Craftsman

Stop using brittle CSS selectors. Use locators that mirror how users see the web.

The Golden Rule: Role-Based Locators

// Best: Reliable and Accessible
await page.getByRole('button', { name: 'Submit' }).click();

// Good: Text-based
await page.getByText('Login successful').isVisible();

// Fallback: Test IDs (industry best practice for QA)
await page.getByTestId('cart-icon').click();

Common Actions

  • 🚀 .fill('text') - Type into input
  • ✅ .check() - Check checkbox/radio
  • 🖱️ .hover() - Mouse hover
  • 📁 .setInputFiles() - File upload

Architectural Excellence

Level 3: The Architect

The Page Object Model (POM)

Maintain large suites by abstracting pages into reusable classes.

// login-page.ts
export class LoginPage {
  readonly page: Page;
  readonly usernameInput: Locator;

  constructor(page: Page) {
    this.page = page;
    this.usernameInput = page.getByPlaceholder('Username');
  }

  async login(user: string) {
    await this.usernameInput.fill(user);
    await this.page.getByRole('button', { name: 'Login' }).click();
  }
}
Advanced Tip

Visual Regression Testing

Capture and compare screenshots automatically to ensure zero visual breakages.

await expect(page).toHaveScreenshot('homepage.png');

Mastering the Trace

Never debug blindly. Use Playwright's time-traveling tools.

The UI Mode

Run tests locally with a live watch mode and step-by-step preview.

npx playwright test --ui

The Trace Viewer

View failures in CI with a full DOM snapshot, console logs, and network traffic.

npx playwright show-trace path/to/trace.zip
